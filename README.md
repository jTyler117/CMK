# CMK
Gives controller control of computers mouse, along with a console style keyboard.

CONTROLLERM+K
================
Use your Xbox controller as a mouse. Works with any Xbox controller
(One / Series / 360), wired or Bluetooth.

INSTALL
-------
1. Extract this whole zip somewhere (don't run it from inside the zip)
2. Double-click install.bat
3. If Windows shows "Windows protected your PC", click
   "More info" -> "Run anyway" (it's just an unsigned app,
   Windows shows this for anything not from a big company)
4. Follow the prompts - it'll make a desktop shortcut and ask if you
   want it to start with Windows

HOW TO USE
----------
- CONTROLLER MODE (default): your controller works totally normally
  for games. The app stays out of the way.
- MOUSE MODE: your stick moves the cursor, and your bindings
  (clicks, keys, scrolling) are active.

First-time setup (takes 2 minutes):
1. Open it from the desktop shortcut
2. Mouse Settings tab: pick which stick moves the mouse
3. Key Bindings tab: "Add Binding" -> "Learn Control" -> press the
   button on your controller -> pick what it does -> Save
4. IMPORTANT: bind one button to "Toggle Mode" (Back/View button is
   perfect) - that's how you switch modes without a keyboard
5. There's also an action to open the Windows on-screen keyboard,
   so you can type with the controller

It lives in the little icon area by the clock (blue C, red M).
Right-click it for quick options. Closing the window just hides it
there - Exit is in that menu.

UNINSTALL
---------
Double-click uninstall.bat

TROUBLESHOOTING
---------------
- "No controller detected": make sure the controller works in a game
  first. If it works in games, it works here.
- Cursor drifts by itself: raise the Deadzone slider.
- Settings are saved automatically and survive restarts.
